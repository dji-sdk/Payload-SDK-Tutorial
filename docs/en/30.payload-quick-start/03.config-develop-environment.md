
This article describes the process of setting up the software environment. After connecting the hardware and setting up the software environment, the developer could create the project and run the sample code.  
This document is intended for payload developers using the DJI Payload SDK.
- If you are new to payload development, it is recommended to read this document in sequence.
- If you have already decided to use Linux / RTOS, you may skip directly to the corresponding section.

### Resource Usage Comparison

#### Linux
Use Manifold 2-C to run PSDK sample program on the Linux platform. Resource usage of the program is as follows:

- Stack: 12288 bytes
- Heap: 40960 bytes
- Text: 755359 bytes
- Data: 3872 bytes
- Bss: 23848 bytes
- CPU: 7.2%

#### RTOS

Using RTOS on the STM32F407IGH6-EVAL, the resource usage of the payload is as follows:

- Text: 353620 bytes
- Data: 1836 bytes
- Bss: 85620 bytes
- CPU: 30%

## Configure Development Environment
Please refer to the corresponding section based on your selected system platform.
### Configure RTOS Development Environment
Download and install the software:

* [Keil MDK](http://www2.keil.com/mdk5/)
    * C Compiler:  Armcc.exe V5.06 update 6 (build 750)
    * Assembler:              Armasm.exe V5.06 update 6 (build 750)
    * Linker/Locator:         ArmLink.exe V5.06 update 6 (build 750)
    * Library Manager:        ArmAr.exe V5.06 update 6 (build 750)
    * Hex Converter:          FromElf.exe V5.06 update 6 (build 750)

* Use Keil    
  - GD32F527 On-Board: After activating Keil MDK software, manually download the latest GD32F527 AddOn package from the [GigaDevice official website](https://www.gd32mcu.com/en/download?kw=f527&lan=en&submit=) and install the GigaDevice.GD32F5xx_DFP.1.3.0.pack driver package.     
  - STM32F4 On-Board：After **activated** the Keil MDK , please use Keil Pack Installer or <a href="http://www.keil.com/dd2/Pack/" target="_blank">download</a> the newest driver of STM32F4xx_DFP.2.x.x.

### Configure Linux Development Environment
Download and install the software:

* C compiler: GCC 5.4.0 / 5.5.0
* CMake: 2.8 and above
* FFmpeg: Version should be equal to or higher than 4.1.3, and lower than 5.0.0
