## Develop And Produce
### Preparations Before Development
Before developing payload devices using the PSDK, it is recommended to first learn the fundamental knowledge required for payload device development, understand the PSDK's capabilities, and familiarize yourself with the standards payload devices must meet.
* Learn drone fundamentals and control principles: such as pitch, yaw, roll, and altitude control
* Review DJI's
  <a href="../payload-quick-start/configure-the-hardware-environment/payload-develop-criterion.html" target="_blank">Payload Standards</a>

### Begin Developing Payload Devices
When developing payload devices using the PSDK, **ensure proper connection** between your selected hardware platform, third-party development board, and DJI drone. **Configure the payload development environment correctly**, import the PSDK properly, and complete initialization setup.
* Before developing payload devices with PSDK, read the
  <a href="../development-preparation/safety-information.html" target="_blank">Safety Information</a>
  to avoid damaging payload devices or drones due to improper operation.
* Connect the drone, hardware platform, third-party development board, and computer.
  * <a href="../payload-quick-start/configure-the-hardware-environment/device-connect.html" target="_blank">Payload Development Hardware Connection</a>
  * <a href="../manifold-quick-start/manifold-connect.html" target="_blank">Manifold Development Hardware Connection</a>
* Configure the development environment and prepare the relevant toolchain and libraries.
  * <a href="../payload-quick-start/config-develop-environment.html" target="_blank">Payload Development: Configure Development Environment</a>
  * <a href="../manifold-quick-start/manifold-development.html" target="_blank">Manifold Development: Configure Development Environment</a>
* Import PSDK and complete initialization.
  * <a href="../payload-quick-start/getting-started-with-psdk.html" target="_blank">Payload Development: Import PSDK</a>
  * <a href="../manifold-quick-start/getting-started-with-psdk.html" target="_blank">Manifold Development: Import PSDK</a>


### Developing Fully-Featured Load Equipment
* Please refer to
  <a href="../function-overview/basic-function/log-management.html" target="_blank">Function Overview</a>
  and
  <a href="https://developer.dji.com/doc/payload-sdk-api-reference/en/" target="_blank">PSDK API Documentation</a>
  to develop your payload device.
* Utilize tools like [DJI Assistant 2](https://www.dji.com/en/downloads) to debug your payload device, refine its functionality, and enhance its performance.

### Apply for Payload Certification and Enable Mass Production Capability
Access the DJI Developer website, navigate to Developer Center -> Apps -> APPLICATION CERTIFICATION.
:::warning Note:
* A single PSDK APP authorization can bind up to 20 payloads. This limit will be removed after applying for mass production capability.
* **Starting January 1, 2025**, newly registered PSDK apps will default to development mode. Ecosystem payloads requiring binding with **Skyport V2, X-Port, or SDK-certified chips** will be subject to this quantity limit in development mode. Ecosystem payloads not involving the binding process remain unrestricted.
:::

## Business Cooperation

Adhere to the concept of openness and mutual benefit, DJI provides developers technical support, payload detection, channel recommendation, and DJI ecological promotion services. We sincerely work with developers and partners to build a good application ecosystem.

- If the payload passed DJI's detection, it will be recommended to third-party partners.
- If payload can pass the inspection of a third-party professional organization, after signing a cooperation agreement with DJI, DJI will carry out more in-depth cooperation with the partner, such as recommending the partner's payload to [Global Users ](https://www.dji.com/en/products/enterprise#partner-payloads), etc. For more cooperation matters, please [Contact Us ](mailto:dev@dji.com).
