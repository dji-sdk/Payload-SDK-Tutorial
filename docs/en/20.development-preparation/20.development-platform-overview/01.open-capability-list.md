

The following table lists the functional support provided by each flight platform when using different interfaces. As there are differences between platforms, please choose the appropriate flight platform and interface according to your functional requirements.

:::tip Note:
- PSDK Interfaces includes one top-mounted gimbal port and two bottom-mounted gimbal ports. For details, please refer to <a href="../../payload-quick-start/configure-the-hardware-environment/device-connect.html#matrice-300-rtk-device-connection" target="_blank">Device Connection</a>.
- The Matrice 400, Matrice 4E, and Matrice 4TD series drones only support basic functions on third-party development platforms. Advanced functions require the use of a Manifold 3.
