## Drone
A drone refers to a flying platform that integrates a flight control system, propulsion system, sensors, communication links, and payload ports, designed to perform flight missions and carry various payload devices.

## Payload Device
A payload device refers to functional modules installed on the drone via ports, such as gimbal cameras, sensors, or actuators, used to fulfill specific business or application functions.

## Flight Platform
The flight platform refers to the complete drone flight system centered on the flight control system, responsible for flight control, attitude stabilization, navigation, positioning, and flight safety.

## Development Platform
A development platform refers to the hardware and software environment used for application or payload development, including the operating system, computing resources, and relevant development ports, such as Manifold‑Linux or Payload Development‑RTOS/Linux platforms.

## Payload Development
Payload development refers to the application development process on the payload side based on the SDK, aimed at enabling communication, control, and data processing between the payload device and the drone system.

## Manifold Development
Refers to the application development method on the Manifold platform based on the Linux system, interacting with the drone platform and payload device via PSDK or other ports to achieve complex computation and business functions.

## Development Suite
The PSDK development suite is a collection of hardware support components for payload development, including development boards, adapter structures, and supporting cables for E‑Port, SkyPort, X‑Port ports, used to validate data communication and power supply capabilities between the payload and drone flight platform.

## Gimbal
A gimbal is a stabilizing device for carrying cameras or sensors, capable of precise multi-axis control to maintain payload attitude stability and ensure controlled shooting direction.

## MSDK
MSDK (Mobile SDK) is a development suite for mobile platforms, supporting functions such as drone control, status retrieval, and mission management in mobile phone or tablet applications.

## Manifold Application
A Manifold application refers to programs running on the Manifold platform, typically developed based on the Linux system, used to execute high computational power tasks or complex business logic.

## Link
A link refers to the communication channel used to transmit data and control information between the drone, payload, and ground station, including both wired and wireless methods.

## Data Subscription
Data subscription is a data interaction method based on the publish‑subscribe model, where applications or payloads can subscribe as needed to flight status, sensor, or system data.

## Stream
A stream refers to video or image data that has been encoded, commonly used for real-time video transmission, processing, or storage.

## Local Upgrade
Local upgrade means completing system or application software updates via storage media or direct connection on the device locally, without the need for remote upgrade over a network.

## Tuning Software Log
Logs generated during the running process of tuning or configuration software on the ground are used to record user operations, parameter configurations, and communication with the drone or payload, mainly for troubleshooting software issues and configuration verification.

## Aircraft Log
Logs generated by the complete drone system during operation and flight are used to record flight status, system events, and abnormal information, serving as a key basis for flight analysis and fault diagnosis.

## Downward Vision
Refers to a visual perception system installed at the bottom of the drone pointing towards the ground, used to obtain ground images or visual information, assisting in drone positioning, hovering, and low-altitude flight control.

## USB-BULK
A type of USB data transmission suited for scenarios requiring high bandwidth but not real-time communication, commonly used for image, video, or file data transfer.

## RNDIS
A USB-based virtual network interface technology that allows devices to establish network connections via USB, enabling upper-layer applications to interact with data using standard network communication methods.

## Mass Production Authorization
Refers to the production license granted to payload products developed with PSDK and successfully validated through relevant tests, supporting the product's entry into mass production and commercial application stages.
