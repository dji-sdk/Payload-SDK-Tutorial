This document systematically introduces the core product capabilities, interface types, and model compatibility of the DJI Payload SDK. It assists developers in quickly understanding and evaluating the PSDK's applicability across different application scenarios from dimensions such as aircraft platforms, interface forms, operating systems, and functional capabilities. This provides a reference basis for payload solution selection and system design.

## Supported Aircraft Models
This section lists the aircraft models currently compatible with PSDK, enabling developers to quickly determine during the solution design phase whether their target flight platform supports payload development via PSDK.
:::tip Notes:
* For detailed instructions on the parameters of the drone and the use of the drone, please refer to the quick start guide of the optional model from [official website](https://enterprise.dji.com).
* The maximum flight time is measured without payloads in a windless environment until the battery level reached 0%. Data is for reference only. Actual usage time may vary depending on the flight mode, accessories, and environment. Please pay attention to reminders in the app.
:::
