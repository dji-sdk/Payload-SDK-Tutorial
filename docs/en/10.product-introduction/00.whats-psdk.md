PSDK (Payload SDK) is a set of software development ports for extending the payload capabilities of drones, enabling developers to integrate self-developed or third-party payloads into the drone platform for control, data interaction, and system coordination. To help you develop safe and reliable payload devices using the new features provided by PSDK, please stay updated with DJI PSDK version releases, and always use the [latest version of the PSDK development kit](https://github.com/dji-sdk/Payload-SDK/releases) for payload development.

> **This article refers to**
>
> * The DJI Mobile SDK is a software development kit designed to give developers access to the capability of DJI's aircraft and handheld camera products. Developers can use mobile apps built with MSDK to control payload devices to perform specified actions and tasks. Click [More](https://developer.dji.com/doc/mobile-sdk-tutorial/en/) to learn about additional MSDK features.
> * "Mobile App" refers to the Mobile App or DJI Pilot developed using MSDK.
> * "Load device" is the load device developed using PSDK.

## PSDK Overview
DJI provides a software interface system centered on PSDK to support developers in creating payload devices that can be mounted on DJI drones, along with hardware components such as the [X-Port Standard Gimbal](https://store.dji.com/cn/product/dji-x-port), [SkyPort V3 Adapter Ring](https://store.dji.com/cn/product/dji-skyport-v3-adapter-set?vid=192891) and [E-Port V2 Coaxial Cable](https://store.dji.com/cn/product/dji-e-port-v2-coaxial-cable-kit?vid=192711). These tools help developers utilize DJI drone resources such as power, communication links, and status information more efficiently. Developers can leverage the functional ports provided by PSDK, combined with specific structural designs, hardware engineering, software logic implementation, and algorithm optimization, to create payload devices tailored for various industry applications, including **automated inspection systems**, **infrared cameras**, **mapping cameras**, **multispectral cameras**, **loudspeakers**, and **searchlights** to meet the needs of different specialized fields.

## Roles and Usage of PSDK in the System
In practical applications, a drone system usually consists of a flight platform, payload devices, onboard computing platforms, and ground applications working in coordination.
As the core port for payload integration and expansion, PSDK can be used in various development forms to enable communication, control, and data interaction between the payload and the drone platform.
Depending on where the application runs, PSDK is typically used in the following two development modes: 
- Payload Development: Directly use PSDK on the payload device to implement payload functions.
- Manifold Development: Use PSDK on the Manifold onboard computing platform to coordinate with both the payload device and the flight platform.

Additionally, PSDK can also be used together with the Mobile SDK (MSDK), allowing the mobile app to perform payload control and task management, forming a complete industry solution.

## Advantages
* **Support for Multiple Development Modes**<br/>
PSDK can be flexibly applied to both Manifold and payload development scenarios, meeting different computing power requirements and system architecture choices.
* **Unified Payload Integration Capability**<br/>
Reduces the integration cost of different types of payloads through a standardized port protocol.

* ****Perfect Function****<br/>
  Using basic functions such as information acquisition, data transmission, and power management and advanced functions such as cameras, gimbals, payload coordination, and precise positioning, developers could design a perfect function payload. And PSDK provides a wealth of interfaces, which is convenient for developers to use third-party applications and algorithm frameworks to develop professional application software using technologies such as image recognition, automatic cruise and SLAM. In addition, it is also convenient for developers to access third-party sensors and cameras or testing equipment to collect the required data information to meet the user's personalized application functions and control needs.

* ****Customized and Scalable****<br/>
  Besides the DJI Pilot is compatible with payload which developed using PSDK, DJI provides Mobile SDK to help developers to develop **Mobile Apps** to control the payload; provides Onboard SDK to support developers write the self-control programs; Windows SDK can help the developer to develop the Data analyzing software.

* **Compatible with a wide range of software and hardware platforms**<br/>
  Applications developed using PSDK can run on mainstream embedded hardware platforms, such as STM32, etc., and can also run on mainstream embedded operating systems and software architectures, such as Linux and RTOS. After performing the required work based on <a href="../payload-quick-start/getting-started-with-psdk.html#cross-platform-porting" target="_blank" rel="noopener noreferrer">Porting</a>, you can run PSDK-based application on different versions of software and hardware platforms.

* ****Support Services****<br/>
  PSDK not only provides the API and hardware for developing the payload, but also provides standards, technical support, marketing, and ecological cooperation services, etc.
