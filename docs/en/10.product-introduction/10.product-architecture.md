
## **PSDK Architecture Overview**

PSDK (Payload SDK) is an open development framework provided by DJI for the payload ecosystem. It enables unified access and collaborative communication between onboard payload devices, the flight platform, the remote controller, and cloud systems. Based on PSDK, payload devices can perform bidirectional communication with the flight platform to obtain flight status and control commands, and can transmit payload data in real time to the remote controller, cloud platforms, or third-party systems.

Within this architecture, PSDK payloads establish connections with the Manifold 3 PSDK application through communication mechanisms such as MOP/IP, enabling payload control, status reporting, and data transmission. Meanwhile, payloads can interact with DJI Pilot on the remote controller and mobile applications based on MSDK via the DJI SDR/4G enhanced video transmission link, supporting Widget-based business control, video stream forwarding, and media management. Through Pilot cloud integration, Dock cloud integration, or custom network communication links, PSDK can also interconnect with third-party cloud platforms and FlightHub 2, fulfilling requirements such as remote monitoring, data aggregation, and business scheduling.

Overall, this architecture establishes a complete communication chain across **PSDK payload devices — onboard computing power / flight platform — remote controller applications — cloud platforms**. It supports multiple communication methods operating in parallel and forms a flexible, scalable, end-to-end payload business interaction system.

## **Manifold 3 Architecture Overview**

Manifold 3 is a high-performance onboard computing platform that supports deep interaction with the flight platform. It enables access to flight control and telemetry data, while also allowing control and management of payloads, including DJI official payloads and PSDK ecosystem payloads. Manifold 3 can communicate with MSDK-based applications and DJI Pilot on the remote controller through the DJI SDR/4G enhanced video transmission link. It also supports connectivity with third-party cloud platforms and FlightHub 2 via Pilot cloud integration or Dock cloud integration. In addition, Manifold 3 can establish custom communication links through onboard 4G or extended 4G/5G CPE network cards.

Manifold 3 further integrates PSDK applications, PSDK payload devices, MSDK-based mobile applications, and cloud platforms based on Cloud API, enabling interconnection across all nodes and forming a complete end-to-end business interaction system.
