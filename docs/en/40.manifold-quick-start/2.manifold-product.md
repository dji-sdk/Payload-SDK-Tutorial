## Manifold 3  Introduction
Manifold 3 is positioned as DJI Industry Drones computing power expansion module, to meet the growing demand for automated operations and AI applications in the industry. It provides 100T computing resources and native PSDK interface support, enabling seamless compatibility and communication with DJI flight platforms. It supports acquiring media and sensor data from flight platforms, flight control, and route planning, enabling more automated operations. Developers can focus on PSDK software development and AI applications without worrying about hardware design and compatibility issues, fully dedicated to creating edge AI applications that meet industry needs. These applications can be packaged into DPK format software packages and distributed to drone users worldwide, opening up new business models.

Supports installing/updating third-party Miaosuan applications on Miaosuan 3 devices through PILOT and Sikong.

**Install/Update third-party applications via PILOT:**<br>
The operation of installing/updating third-party applications through PILOT is as follows. After obtaining the DPK application installation package, copy the installation package file (i.e. xxx.dpk file) to the remote control through a USB cable or SD card, open the PILOT Miaosuan 3 interface, and click **Application Management**to enter the Miaosuan 3 application management page.

**Install/Update third-party applications via FLIGHTHUB 2:**<br>
The process for installing/updating third-party apps via FLIGHTHUB 2 involves the following steps:

1. Log in to [FLIGHTHUB 2](https://es-flight.dji.com/login) using a web browser.
2. Access the Devices interface.
3. Search for and select the desired device.
4. Choose the "Device Maintenance" option for that device.
5. Navigate to the "Manifold 3 Application Management" within the Device Maintenance.
6. In the application management interface, click "+ Import" (located in the upper right corner).
7. Select the xxx.dpk file to install or update the DPK application.

### Third-Party App Uninstall && Data Clearance & Log Export
Support to uninstall third-party apps, clear app data, and export app logs via PILOT and FLIGHTHUB 2.

- **Uninstall**: In the Manifold 3 interface, select the app Details -> Uninstall.
- **Clear Data**: Select "Clear Data" to remove app runtime-generated data.
 Log export: After entering the Manifold 3 application management interface, you can export the logs of third-party applications to facilitate developers to conduct post-sales analysis of problems.
  - Export Logs via PILOT: insert the USB flash drive or card reader into the universal USB port of Miosuan, and then click the "Log Export" option on PILOT to export the application logs on Miosuan to the USB flash drive.
  - Export Logs via FLIGHTHUB 2: select the application you want to export logs and click the "Log Export" option to export the application logs to the local.

### Modify Boot Auto-Start Application
1. Support modifying the boot auto-start application on PILOT. The operation in PILOT is: Click Manifold 3 -> Application Management -> More of each application -> Set to Boot Auto-Start to modify the current application to auto-start on boot.
2. Applications that have been set to auto-start can be disabled, and the operation is the same as above.
3. Only one application can be set to auto-start on boot. The last set application will have the auto-start priority.

### Firmware Upgrade
Support to upgrade Manifold 3 firmware via DJI Assistant 2 or DJI PILOT APP (recommended: DJI Assistant 2).

### File Export

| Method               | Description |
|----------------------|-------------|
| USB Export           | Connect USB flash drive/SSD to the Manifold's USB-C port, then use PILOT APP to export app data with one click. |
| Computer Export      | With Manifold mounted on the aircraft, connect to a computer via the aircraft's debugging port. In DA2's log export interface, click "Open Device Directory" and export files via Windows Explorer. |
| Debug Export (Dev)   | Developers can directly export files via SSH/SFTP. |

### SSH Commands for App Management
Support app operations such as installing, starting, stopping via SSH commands, as follows:

```bash
# Install/Update app
dji_app_ctl install -i xxx.dpk
# Uninstall app
dji_app_ctl uninstall xxx(app_name)
# Start app
dji_app_ctl start xxx(app_name)
# Stop app
dji_app_ctl stop xxx(app_name)
# Stop app
dji_app_ctl stop xxx(app_name)
# Restart app
dji_app_ctl stop xxx(app_name)

# List all apps info
dji_app_ctl list
# List all app running status
dji_app_ctl status

# List all name=app_name app info
dji_app_ctl list xxx[app_name]
# List all name=app_name app running status
dji_app_ctl status xxx[app_name]
```
