## Hardware Environment Preparation

Manifold 3 provides two power supply methods:
Connect to the drone (M400 drone E-Port V2 interface / Matrice 4D series drone E-Port interface) for power supply, or use the USB expansion dock PD power port to connect to a power adapter (65W/100W).
- The USB-C port on the MiaoSuan 3 supports connecting to a docking station for setting up a development environment (does not support display output). Recommended models: Ugreen CM512 Type-C Docking Station, Anker (A8380) 553PowerExpand 8-in-1 USB-C Hub.
>- For Matrice 4D/4TD series aircraft using Connection Method 1, wiring to the Manifold 3 must be completed before aircraft startup. This connection method does not support hot-swapping.

When connecting to the drone via E-Port V2, users can directly connect the PC and the drone's configuration port using a USB cable, and access Manifold 3 through the USB virtual network card by visiting the corresponding IP addresses of different E-Port V2 interfaces.
When using the USB expansion dock, users can connect to the local area network via an Ethernet cable and then SSH into Manifold 3 using the IP address assigned by the router.

<br>In both methods, users can access Manifold 3 for development and debugging via SSH or VNC.
