## Supported External Devices

Manifold 3 is equipped with a USB-C port that supports OTG cables and USB-C docking stations (USB hubs). It can be used to connect various USB Type-A peripherals and also supports PD power input. The supported device types and reference models are listed below:

<table style="border-collapse:collapse; width:100%;">
    <thead>
        <tr style="background:#f2f2f2;">
            <th style="text-align:left;">Device Type</th>
            <th style="text-align:left;">Description / Reference Models</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Docking Station</td>
            <td>
                Supports connection to a USB-C docking station for USB peripheral expansion, USB Ethernet access, and PD power input through the dock.
            </td>
        </tr>
        <tr>
            <td>PD Power Adapter</td>
            <td>
                The USB-C port supports PD power input.<br>
                Reference models: DJI 65W Portable Charger, DJI 100W Desktop Charger, Apple 61W USB-C Power Adapter, Apple 140W USB-C Power Adapter.
            </td>
        </tr>
        <tr>
            <td>USB Storage Devices</td>
            <td>
                Supports connecting USB flash drives, USB SSDs, and other storage devices through the USB-C port. Compatible file systems include exFAT, NTFS, and FAT32.
            </td>
        </tr>
        <tr>
            <td>USB Ethernet Adapters</td>
            <td>
                Supports USB Ethernet adapters connected through OTG for wired network access.<br>
                Reference chipsets: AX88179A, RTL8152.
            </td>
        </tr>
        <tr>
            <td>USB Wi-Fi Adapters</td>
            <td>
                Supports USB Wi-Fi adapters connected through OTG to extend wireless connectivity.<br>
                Reference models: RTL8852BU Wi-Fi 6 adapter, RTL88X2BU Wi-Fi 5 adapter.
            </td>
        </tr>
        <tr>
            <td>USB-to-Serial Modules</td>
            <td>
                Supports USB-UART modules connected through OTG for serial communication and debugging.<br>
                Reference models: FT232, CP2102.
            </td>
        </tr>
        <tr>
            <td>USB CPE Devices</td>
            <td>
                Supports external 4G/5G CPE devices, provided that the device supports the USB RNDIS protocol.<br>
                Reference models: CPE devices based on the UNISOC T760 platform.
            </td>
        </tr>
    </tbody>
</table>
