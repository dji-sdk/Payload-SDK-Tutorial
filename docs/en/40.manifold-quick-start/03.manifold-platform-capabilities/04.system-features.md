## Accessing Internet
This function allows Manifold 3 applications to access the Internet through the aircraft's onboard 4G network, requiring the aircraft to have DJI Enhanced Transmission Module installed and corresponding data package activated.
By default, the payload network access function is already enabled without any operation required (dependent on Enhanced Transmission Module being installed on the aircraft and 4G network function being normal). Supports Matrice 400, Matrice 4TD/4D, and Matrice 4T/4E aircraft.
For more detailed function descriptions, please refer to <a href="../../function-overview/advanced-function/access-internet.html" target="_blank" rel="noopener noreferrer">Access Internet</a>.
## High Power Output
High-voltage output can be enabled on the rear USB‑C port of Manifold 3 through the PSDK API. To avoid damage, do not connect standard USB peripherals, such as USB flash drives, when high-voltage output is enabled.  
PSDK API Interface: `DjiPowerManagement_OutputHighPower`
- Connected to Matrice 400: 14V / 5A 
- Connected to Matrice 4E/4T: 12.8V-17V / 3.38A
- Connected to Matrice 4D/4TD: 16.8V-25.5V / 3A
## Power Adaptive
Manifold 3 adopts a graded power consumption management strategy, automatically balancing power consumption and performance according to system operating status to meet the long-term operation requirements of industry drones.
- Standby State: When no application business is running or system load is low, the system adopts a low power consumption strategy, limiting CPU usage to 4 CPU cores and adaptively adjusting frequency according to load, minimizing power consumption while ensuring stable operation of basic functions.
- Running State: When application business starts or load increases, the system automatically switches to MAXN (maximum performance) mode, enabling all 8 CPU cores and simultaneously releasing the full performance capabilities of CPU, GPU, and DDR. NVIDIA platform dynamically schedules module frequencies based on real- time load to ensure stable business operation.
Power mode switching is automatically completed by the system without developer intervention, allowing developers to focus on application function and business function development.
