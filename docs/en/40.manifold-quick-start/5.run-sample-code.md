The latest DJI PSDK (Payload SDK) development kit includes sample code for Manifold applications. Follow these steps to compile and run the sample program to quickly understand and experience Manifold application features.

## 1. Modify Required Configuration Code

You need to create an application in the Developer User Center, then fill the generated Payload SDK-Manifold 3 application information into specified files in the PSDK development kit. Otherwise, the sample application won't run properly.

* `samples/sample_c/platform/linux/manifold3/application/dji_sdk_app_info.h`
* `samples/sample_c++/platform/linux/manifold3/application/dji_sdk_app_info.h`

```c
/* Exported constants --------------------------------------------------------*/
// ATTENTION: User must go to https://developer.dji.com/user/apps/#all to create your own DJI SDK application, get DJI SDK application
// information then fill in the application information here.
#define USER_APP_NAME               "your_app_name"
#define USER_APP_ID                 "your_app_id"
#define USER_APP_KEY                "your_app_key"
#define USER_APP_LICENSE            "your_app_license"
#define USER_DEVELOPER_ACCOUNT      "your_developer_account"
#define USER_BAUD_RATE              "460800"
```

The sample code already includes hardware link adaptation for Manifold 3, user don't need to modify again.

## 2. Compile and Experience the Sample
The PSDK package supports CMake compilation. Follow these steps:

1. Modify the CMakeLists.txt file, changing:

```cmake
add_subdirectory(samples/sample_c/platform/linux/manifold2)
add_subdirectory(samples/sample_c++/platform/linux/manifold2)
```

to:

```cmake
add_subdirectory(samples/sample_c/platform/linux/manifold3)
add_subdirectory(samples/sample_c++/platform/linux/manifold3)
```

2. Use the cmake command to compile and generate executable files. After compilation, the following executables will be generated in the build/ directory:

- dji_sdk_demo_on_manifold3
- dji_sdk_demo_on_manifold3_cxx

3. Run the executables to experience the sample code.

```c
cd Payload-SDK/
mkdir cmake-build-debug && cd cmake-build-debug
cmake ../  && make -j
./bin/dji_sdk_demo_on_manifold3
```
