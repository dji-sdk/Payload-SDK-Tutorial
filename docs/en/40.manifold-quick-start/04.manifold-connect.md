Manifold 3 provides the following two power supply and connection methods:
1. **Powered by the Aircraft**
   - Matrice 400: E-Port V2 port  
   - Matrice 4D: E-Port port
2. **Powered via a USB Hub PD Port**  
   Connect a power adapter (65W/100W) through a USB hub PD port.

::: tip Notes:
- The USB-C port of Manifold 3 supports connecting a USB hub to set up the development environment (display output is not supported). Recommended models include: UGREEN CM512 Type-C Hub, Anker (A8380) 553 PowerExpand 8-in-1 USB-C Hub.
- For Matrice 4D/4TD series aircraft, when using connection method 1, the wiring between the aircraft and Manifold 3 must be completed before powering on the aircraft. Hot-plugging is not supported with this connection method.
- When connecting to the aircraft via E-Port V2, users can directly connect a PC to the aircraft tuning port using a USB cable and access Manifold 3 via different IP addresses corresponding to different ports through a USB virtual network adapter.
- When using a USB hub connection, users can connect to a local area network via Ethernet and access Manifold 3 by logging in via SSH using the IP address assigned by the router.
- In both connection methods, users can access Manifold 3 via SSH or VNC for development and debugging.
:::
