## Connecting Manifold 3 to an External Network

This connection method corresponds to **Connection Method 2** in the development environment topology diagrams for Matrice 400, Matrice 4D / 4TD, and Matrice 4E / 4T. Manifold 3 uses a USB docking station to connect external PD power and a network adapter. Users can access Manifold 3 through a LAN connection via Ethernet cable or USB Wi‑Fi, using the IP address assigned by the router.
- For the first access, the IP address of Manifold 3 can be obtained either from the router management page or through Connection Method 1.
- To configure a static IP address, set a fixed IP on the router or use the <a href="../manifold-platform-capabilities/system-tools.html#network-configuration-tool" target="_blank" rel="noopener noreferrer">`netctl.sh` network configuration tool</a> on Manifold 3. 

## Direct Connection Between Manifold 3 and a PC

Manifold 3 can be connected to a USB docking station or a USB Ethernet module, and then directly connected to the Ethernet port of a Windows PC through an RJ45 cable. (The PC must have two Ethernet interfaces, or the PC must access the Internet through WLAN.)

In Windows, open:  
**Control Panel > Network and Internet > Network Connections**

<img src="https://terra-1-g.djicdn.com/71a7d383e71a4fb8887a310eb746b47f/psdk/%E5%A6%99%E7%AE%97%E5%BC%80%E5%8F%91/%E7%A1%AC%E4%BB%B6%E7%8E%AF%E5%A2%83%E6%90%AD%E5%BB%BA/%E5%A6%99%E7%AE%973%E4%B8%8EPC%E7%9B%B4%E8%BF%9E.png"  style="width: 50%;" />

As shown in the figure, at least two network adapters should be visible. One is the network adapter that the PC uses for normal Internet access, and the other is the network adapter connected to Manifold 3.

Ethernet sharing enables communication between Manifold 3 and the PC, and provides Manifold 3 with Internet access. Right-click the network adapter on the PC that is already connected to the Internet, select **Sharing**, enable Internet sharing, and choose the network adapter connected to Manifold 3.
