## What Is Manifold 3

Manifold 3 is a computing expansion module designed for DJI enterprise drones. It provides high-performance onboard computing resources and native support for the PSDK interface, enabling seamless compatibility and stable communication with DJI flight platforms. It supports access to media and sensor data from the flight platform and, combined with flight control and waypoint planning capabilities, helps developers build automated and intelligent applications.

Manifold 3 provides unified abstraction and encapsulation of the underlying hardware and system capabilities, allowing developers to focus on PSDK software and AI application development. Applications can be packaged and deployed in DPK format for standardized installation and distribution, enabling efficient delivery of industry-specific solutions.

## Core Concepts

Manifold development focuses on modularizing and componentizing complex computing and data processing capabilities, allowing developers to quickly leverage powerful computing resources without needing to focus on low-level implementation details.

Its core concepts are mainly reflected in the following aspects:
- **Computing Power as a Service**: Computing capabilities are provided in the form of services and are ready to use out of the box.
- **Low-Barrier Development**: Configuration-based and template-based approaches reduce the amount of code required.
- **Modular Composition**: Decoupled capabilities that can be flexibly combined.

## Manifold 3 Architecture Overview

Manifold 3 is a high-performance onboard computing platform that supports deep interaction with the flight platform. It enables access to flight control and telemetry data, while also allowing control and management of payloads, including DJI official payloads and PSDK ecosystem payloads. Manifold 3 can communicate with MSDK-based applications and DJI Pilot on the remote controller through the DJI SDR/4G enhanced video transmission link. It also supports connectivity with third-party cloud platforms and FlightHub 2 via Pilot cloud integration or Dock cloud integration. In addition, Manifold 3 can establish custom communication links through onboard 4G or extended 4G/5G CPE network cards.

Manifold 3 further integrates PSDK applications, PSDK payload devices, MSDK-based mobile applications, and cloud platforms based on Cloud API, enabling interconnection across all nodes and forming a complete end-to-end business interaction system.
