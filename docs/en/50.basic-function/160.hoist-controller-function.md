## Overview

Built on a third‑party PSDK payload paired with DJI hoist system 2.0, this solution provides simple control over the hoist system and real‑time operational feedback.

### Compatible with the Flycart100.

* Target hardware: one E‑Port Lite interface (PSDK) and one Flycart aircraft
* Application scope: System status of the aerial‑hook (sling‑load) system can be retrieved via the E‑Port Lite port using the Payload SDK (PSDK).
* Application constraints: Control of hoist sling‑load functionality and advanced/state‑level status acquisition are not supported.

## vFunctionality overview
By leveraging the Payload SDK (PSDK), the system supports simple control of the hoist system. Custom development on the PSDK enables control of DJI's hoist solution.

## Control of the hoist functionality using the Payload SDK (PSDK).

#### 1. Initialization of the aerial‑hook control module.
Prior to using the "Hosit Control" functionality, initialize the corresponding module to guarantee correct operation.

```c
returnCode = DjiAircraftInfo_Init();
if (returnCode != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS) {
    DJI_ERROR_PRINT_MSG(TAG_NAME_CORE, returnCode);
    DJI_LOG_ERROR(TAG_NAME_CORE, "Aircraft info init error");
    return returnCode;
}
```

#### Aircraft confirmation

**Recommended**  Get the model via the model‑fetch function exposed by the Payload SDK (PSDK) library.

```c
returnCode = DjiAircraftInfo_GetBaseInfo(&aircraftBaseInfo);
if (returnCode != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS) {
    DJI_LOG_ERROR(TAG_NAME_CORE, "Get aircraft base info error");
    return returnCode;
} else {
    DJI_LOG_INFO(TAG_NAME_CORE, "Identify AircraftType = %s, MountPosition = %s, SdkAdapterType = %s",
                    DjiAircraftInfo_GetAircraftName(aircraftBaseInfo.aircraftType),
                    DjiAircraftInfo_GetMountPositionName(aircraftBaseInfo.mountPosition),
                    DjiAircraftInfo_GetAdapterName(aircraftBaseInfo.djiAdapterType));
}
```

#### 2. Start real‑time status acquisition and control for the hoist system.

Enable task execution. The current recommendation is that the task frequency should not exceed 10 Hz.

1. Send a control request to the aircraft for the hoist system. Note: Ground control is not supported — takeoff must be initiated via the APP simulator.  

```c
djiStat = psdk_HoistController_NotifyFlight_to_startPushInfo();
if (djiStat != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS) {
    USER_LOG_ERROR("get hoist can start contrller error: 0x%08llX.", djiStat);
    continue;
}
```

2. Start control hoist system

Hoist winch controller interface:

```c
T_DjiReturnCode psdk_hoistControllerCmdAction(E_DjiHoistControllerCmdStatus cmd)
```

Hoist hook controller interface:
```c
T_DjiReturnCode psdk_hookControllerCmdAction(E_DjiHookControllerCmdStatus cmd)
```

3. Get hoist system return status:

```c
static T_DjiReturnCode subscriptionHoistAndHookStatusData(void)
{
    T_DjiReturnCode djiStat;
    static uint32_t cnt = 0;

    cnt++;
    djiStat = DjiHoistController_hoistStatus(&hoist_status); //Hoist winch status
    if (djiStat != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS){
        return djiStat;
    }

    if (cnt % 5 == 0){
        USER_LOG_INFO("hoist status:cab_s:%d ,can_s:%d, g_w:%d, mg_w:%d, c_s_s:%d, cc_l:%d",
            hoist_status.cable_state, hoist_status.can_status, hoist_status.goods_weight, hoist_status.max_goods_weight,
            hoist_status.cargo_swing_speed, hoist_status.cur_cable_length);
        USER_LOG_INFO("hoist status: c_f:%d, c_an:%d, is_r_t:%d, is_rmr:%d, cdcs:%d",
            hoist_status.cable_force, hoist_status.cable_angle, hoist_status.is_reach_top, hoist_status.is_reach_max_round,
            hoist_status.cable_direction_check_state);
    }

    djiStat = DjiHoistController_hookStatus(&hook_status); //hoist hook stauts
    if (djiStat != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS){
        return djiStat;
    }
    if (cnt % 5 == 0){
        USER_LOG_INFO("hook status: steer_state:%d, weight:%d, warning:%d",
            hook_status.steer_state, hook_status.cargo_weight, hook_status.cargo_warnig);
        USER_LOG_INFO("hook status: error:%d, ctr_state:%d", hook_status.cargo_error, hook_status.ctrl_state);
    }

    return djiStat;
}
```
> **Instruction:** This feature is currently supported only on the Flycart 100.
