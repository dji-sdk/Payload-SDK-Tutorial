## Overview
Transport mode development based on a PSDK payload used with Matrice 400 allows developers to configure the transport mode of Matrice 400, enabling safer and more reliable long-distance transport operations.

### Usage on Matrice 400
- **Applicable configuration:** One PSDK device and one Matrice 400 aircraft. Ports E1 to E4 support connection to E-Port V2 and SkyPort V3. If an expansion hub is connected to the E4 port, up to 7 ports can be supported for E-Port V2 development boards.
- **Supported capabilities:** Transport mode can be configured through PSDK.
- **Constraints:** The PSDK device must be certified. The dimensions of the transport box mounted on Matrice 400 must comply with DJI official requirements to ensure safe and reliable system operation. For specific size requirements, please contact DJI.

## Using the Transport Control Feature

### Setting the Payload Type
Developers can use this API to set the payload type of the PSDK device so that the aircraft enters Transport mode.

- **Supported product:** Matrice 400  
- This setting is required when the payload type is **transport gear**, and `DjiCore_SetSdkDeviceType(DJI_SDK_DEVICE_TYPE_TRANSPORT_BOX);` must be called before `DjiCore_Init`.

```cpp
T_DjiReturnCode DjiCore_SetSdkDeviceType(E_DjiSdkDeviceType type);
```
