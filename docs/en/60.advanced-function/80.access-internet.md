### Environment Preparation
1. Prepare M400 aircraft, Maosuan 3, DJI Enhanced Transmission module, and DJI remote controller.
2. Install the DJI Enhanced Transmission module on the M400 aircraft to ensure that the 4G network function is working properly.
3. Connect Maosuan 3 to the EPort of the M400 (supports E1, E2, E3).
> **Note:**
> 1. This function is currently supported only by Maosuan 3.

### Usage
#### 1. Maosuan 3 Supports Network Access by Default
By default, the payload network access function is enabled and no additional operations are required (this depends on the Enhanced Transmission module being installed on the aircraft and the 4G network function being normal).
Supports Matrice 400, Matrice 4TD/4D, Matrice 4T/4E aircraft.

### Internet Access Troubleshooting
 1. Ensure that the aircraft is connected to the Enhanced Transmission module and the 4G network is normal. The dongle module's green light should be constantly on (you can also check the network configuration on the remote controller).
 2. Ensure that the aircraft is connected to Maosuan 3 and the connection is good. If the connection is poor, reconnect it.
 3. Restart the aircraft and Maosuan 3 and try again.
 4. If the above steps do not solve the issue, please consult DJI Technical Support.

### Interface for Querying Enhanced Transmission State
The interface for querying the Enhanced Transmission state allows you to check the Enhanced Transmission state.

#### Interface Description
```c
/**
 * @brief Get Enhanced Transmission state, which is set in the mobile app image transmission setting -- Enhanced Transmission.
 * @param state: Enhanced Transmission state.
 * @return Execution result.
 */
T_DjiReturnCode DjiAircraftInfo_GetEnhancedTransmission(E_DjiEnhancedTransmissionState *state);

/**
 * @brief Enhanced transmission state.
 */
typedef enum {
    DJI_ENHANCEED_TRANSMISSION_STATE_DISABLED = 0, /**< Mobile app image transmission setting, Enhanced transmission disabled */
    DJI_ENHANCEED_TRANSMISSION_STATE_ENABLED  = 3, /**< Mobile app image transmission setting, Enhanced transmission enabled. */
} E_DjiEnhancedTransmissionState;
```

Based on the obtained Enhanced Transmission state, you can use custom HMS messages to display custom HMS alerts on the pilot. For example, if you find that the network is poor while the Enhanced Transmission is enabled, you can display an HMS alert. Refer to the following, and for specific usage, see [Custom HMS Function](https://developer.dji.com/doc/payload-sdk-tutorial/en/basic-function/hms-customization.html)
