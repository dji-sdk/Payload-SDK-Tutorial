### 资源占用
#### Linux
使用Manifold 2-C 运行Linux 平台PSDK 示例程序，程序运行时的资源占用情况如下所示：
* 栈：约 12288 字节
* 堆：约 40960 字节
* Text 段：755359 字节
* Data 段：3872 字节
* Bss 段：23848 字节
* CPU 占用：7.2 %

#### RTOS 
使用STM32F407IGH6-EVAL 运行RTOS 平台PSDK 示例程序，程序运行时的资源占用情况如下所示：
* Text 段：353620 字节
* Data 段：1836 字节
* Bss 段：85620 字节
* CPU 占用：30 %
