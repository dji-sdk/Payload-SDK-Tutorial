### 环境准备
1. 准备M400飞机、妙算3、DJI增强图传模块、DJI遥控器。
2. DJI增强图传模块安装在M400飞机上，确保4G网络功能正常。
3. 妙算3接在M400的EPort口（支持E1\E2\E3）。
> **说明：**
> 1. 此功能目前仅妙算3支持

### 使用方法
#### 1. 妙算3上线后默认支持
默认情况下，负载网络访问功能已经开启，无需进行任何操作（依赖增强图传模块安装在飞机上，且4G网络功能正常）。
支持Matrice 400、Matrice 4TD/4D、Matrice 4T/4E 三款飞机

### 上网故障排查
1. 确保飞机已经连接上增强图传模块，且4G网络正常，dongle模块绿灯应常亮（也可再遥控器上查看网络配置是否正常）
2. 确保飞机已经连接上妙算3，且接触良好，如果接触不良，请重新连接
3. 重启飞机，重启妙算3后重试
4. 如果以上步骤均无法解决，请咨询DJI技术支持


### 提供查询增强图传开关状态接口
支持查询增强图传开关状态接口，通过接口查询增强图传开关状态。

#### 接口说明
```c
/**
 * @brief Get Enhanced Transmission state, which is set in the mobile app image transmission setting -- Enhanced Transmission.
 * @param aircraftVersion: Enhanced Transmission state.
 * @return Execution result.
 */
T_DjiReturnCode DjiAircraftInfo_GetEnhancedTransmission(E_DjiEnhancedTransmissionState *state);

/**
 * @brief enhanced transmission state.
 */
typedef enum {
    DJI_ENHANCEED_TRANSMISSION_STATE_DISABLED = 0, /**< Mobile app image transmission setting, Enhanced transmission disabled */
    DJI_ENHANCEED_TRANSMISSION_STATE_ENABLED  = 3, /**< Mobile app image transmission setting, Enhanced transmission enabled. */
} E_DjiEnhancedTransmissionState;
```

根据获取到的增强图传状态，可通过自定义hms消息，在pilot上进行自定义hms告警显示；比如当发现网络不好的时候，而此时增强图传又是打开的，那就可以进行hms提示, 参考如下，具体使用参考 [自定义 HMS 功能](https://developer.dji.com/doc/payload-sdk-tutorial/cn/basic-function/hms-customization.html)
