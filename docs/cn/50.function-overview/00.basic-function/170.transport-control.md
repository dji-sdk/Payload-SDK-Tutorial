## 概述

基于 PSDK 负载与 Matrice 400 搭配进行运载模式开发，可设置 Matrice 400 的运载模式，从而实现更安全、更可靠的长距离运输作业控制。

### 在 Matrice 400 上使用

- **使用对象：** 1 个 PSDK 设备和 1 台 Matrice 400 飞行器。其中，E1 至 E4 接口支持连接 E-Port V2 和 SkyPort V3；若在 E4 接口外接扩展坞，则 E-Port V2 开发板最多可扩展支持 7 个接口。
- **使用范围：** 可通过 PSDK 设置运载模式。
- **使用约束：** PSDK 设备需通过认证。Matrice 400 搭载的运载箱尺寸请遵循 DJI 官方要求，以确保系统处于安全、可靠的运行状态，具体尺寸联系 DJI 获取。

## 使用运载控制功能
### 设置负载类型

开发者可以通过该接口设置 PSDK 设备的负载类型，以使飞行器进入运载模式。

- 支持产品：Matrice 400  
- 当负载类型为**运载脚架**时需要进行设置，并且必须在执行 `DjiCore_Init` 之前调用 `DjiCore_SetSdkDeviceType(DJI_SDK_DEVICE_TYPE_TRANSPORT_BOX);`。

```cpp
T_DjiReturnCode DjiCore_SetSdkDeviceType(E_DjiSdkDeviceType type);
```
