

## 应用的安装与更新

## 应用启动与停止
DPK 应用安装或更新完成后，可对应用进行启动、停止操作，并查看其当前运行状态。支持在 DJI Pilot 和司空中启动或停止应用，并查看应用的运行状态。

### 使用 PILOT 启动/停止应用
进入 PILOT APP，点击 **进入飞行界面**。

界面左侧会显示当前妙算 3 设备中已安装/更新的相关应用。点击任一应用图标即可启动应用；应用启动成功后，对应图标会变为黄色，同时飞行界面会显示应用运行时的场景，您可据此测试应用功能。再次点击该应用图标可停止应用，对应图标恢复为灰色，表示应用已停止运行。

## 应用卸载
如果不再需要已安装或更新完成的第三方 DPK 应用，可将其从妙算 3 设备中卸载。操作步骤如下：进入妙算 3 应用管理界面，选择需要删除的应用，然后点击“应用卸载”，即可删除对应应用。

## 应用数据清除
第三方应用在运行过程中会产生日志文件及各类数据。如需清除这些运行数据，可在妙算 3 应用管理界面中选择“数据清除”。


## 应用日志导出
进入妙算 3 应用管理界面后，可导出第三方应用日志，便于开发者进行问题定位和售后分析。

- **在 Pilot 上导出日志：** 将 U 盘或读卡器插入妙算的通用 USB 接口，然后在 Pilot 中点击“日志导出”，即可将妙算上的应用日志导出到 U 盘。
- **在司空上导出日志：** 选择需要导出日志的应用，点击“日志导出”，即可将应用日志导出到本地。
- **支持的 U 盘或 SD 卡格式：** exFAT、NTFS、FAT32。

## 文件导出功能
| 导出方式                                   |说明|
| :-------------------------------------------------------- |:-------------------------------------- |
| U 盘导出方式                    | <ul><li>通过将 U 盘或 SSD 固态硬盘连接至妙算的 USB‑C 接口，可在遥控器的 DJI Pilot App 中一键导出应用数据。</li><li>U 盘或者 SD 卡支持格式：exFAT, NTFS, FAT32。</li></ul>|
| 电脑导出方式                   | 妙算挂载在飞机上，通过飞机的调参口连接电脑，在 DA2的日志导出界面点击“打开设备目录”，然后在 windows 上通过文件资源管理器导出文件。|
| 调试导出方式（开发者功能）       | 开发者可以直接通过 ssh 和 sftp 导出文件数据|

## 修改开机自启动应用
- 支持在 PILOT 上进行修改应用开机自启动功能，在 PILOT 上的操作为：点击 妙算3 > 应用管理 > 每个应用的更多 > 设为开机自启动选项，即可修改当前应用为开机自启。
- 已经设置为自启的应用支持取消开机自启动，操作方式同上。
- 仅支持一个应用进行开机自启动，后设置的应用拥有开机自启动权。

:::tip 说明：
以上方式仅支持 DPK 应用设置开机启动，若您不需要对妙算3的 DPK 应用进行软件分发，或者您的应用依赖类似 ROS 的复杂环境/不方便打包成 DPK，可以参考 <a href="../../faq.html#如何使用-systemd-user-配置开机自启动" target="_blank" rel="noopener noreferrer">使用 systemd --user 方法设置开机启动</a>。
:::
