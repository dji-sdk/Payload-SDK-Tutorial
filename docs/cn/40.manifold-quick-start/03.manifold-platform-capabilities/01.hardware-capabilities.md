## 外部设备支持

妙算 3 配备 USB‑C 接口，支持 OTG 连接线 及 Type‑C 扩展坞（USB‑Hub），可扩展多种 USB Type‑A 外设，并支持 PD 供电输入。支持的设备类型及参考型号如下：
<table style="border-collapse:collapse; width:100%;">
    <thead>
        <tr style="background:#f2f2f2;">
            <th style="text-align:left;">设备类型</th>
            <th style="text-align:left;">功能说明 / 参考型号</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>扩展坞</td>
            <td>
                支持连接 USB‑C 扩展坞，实现 USB 外设扩展、USB 以太网接入，并支持通过扩展坞进行 PD 供电输入。
            </td>
        </tr>
        <tr>
            <td>PD 电源适配器</td>
            <td>
                USB‑C 接口支持 PD 协议电源输入。<br>
                参考型号：DJI 65W 便携充电器、DJI 100W 桌面充电器、Apple 61W USB‑C 电源适配器、Apple 140W USB‑C 电源适配器
            </td>
        </tr>
        <tr>
            <td>USB 存储设备</td>
            <td>
                支持通过 USB‑C 接口连接 U 盘、USB SSD 等存储设备，兼容 exFAT、NTFS、FAT32 文件系统。
            </td>
        </tr>
        <tr>
            <td>USB 以太网卡</td>
            <td>
                支持通过 OTG 方式连接 USB 以太网卡，实现有线网络接入。<br>
                参考型号：AX88179A、RTL8152
            </td>
        </tr>
        <tr>
            <td>USB Wi‑Fi 网卡</td>
            <td>
                支持通过 OTG 方式连接 USB Wi‑Fi 网卡，扩展无线网络能力。<br>
                参考型号：RTL8852BU Wi-Fi 6无线网卡、RTL88X2BU Wi-Fi 5无线网卡
            </td>
        </tr>
        <tr>
            <td>USB 转串口模块</td>
            <td>
                支持通过 OTG 方式连接 USB‑UART 模块，用于串口通信与调试。<br>
                参考型号：FT232、CP2102
            </td>
        </tr>
        <tr>
            <td>USB CPE 设备</td>
            <td>
                支持外接 4G/5G CPE 设备，需设备支持 RNDIS 网络协议。<br>
                参考型号：基于紫光展锐 T760 平台的 CPE 设备
            </td>
        </tr>
    </tbody>
</table>
