## 什么是妙算应用？
妙算应用是基于 DJI PSDK 接口，面向妙算 3 平台开发的应用软件，简称为妙算应用（Manifold Application），其标准分发形态为 DPK 应用。
开发者可通过交叉编译方式，将基于 PSDK 开发完成的应用，使用 DJI 提供的应用打包工具 build_dpk.sh 打包生成以 .dpk 为后缀的安装包（即妙算应用安装包）。最终用户可通过 DJI Pilot App 或 妙算 3 设备内置的应用管理工具 dji_app_ctl，完成妙算应用的安装、更新和卸载。
通过 DPK 应用形态，妙算 3 将应用部署和管理过程进行了标准化封装，相比妙算 2 无需用户掌握 Linux 相关指令，显著降低了现场部署和使用门槛，提升了行业应用的易用性与可交付性。
## 妙算与 PSDK 的关系
PSDK 是妙算应用与 DJI 飞行平台之间的接口层，负责开放飞行平台的能力；妙算 3 则为 PSDK 应用提供算力、系统环境和运行载体。开发者基于 PSDK 开发应用，并运行和部署在妙算 3 上，从而实现与飞行平台的交互与控制。

## 妙算应用开发规范与指南

开发妙算应用需基于 DJI PSDK，并遵循妙算 3 平台的应用开发与部署规范。为确保应用能够正常打包、安装及运行，开发者需满足以下开发约束，并遵循相关开发建议。

### 开发约束

* **依赖库限制**  
  妙算应用仅支持使用**静态链接库**（Static Library），不支持动态链接库（Dynamic Library）。  
  如使用动态库，可能导致应用打包、安装或启动失败。

* **配置文件路径规范**  
  应用源码中涉及配置文件的读写操作，路径必须使用**相对路径**，禁止使用绝对路径。  
  否则可能引发应用安装失败或启动异常。

* **app.json 配置文件要求**  
  开发者需按照规范创建应用级配置文件 `app.json`。  
  可参考 DJI PSDK 软件开发包中的示例文件：`samples/sample_c/platform/linux/manifold3/app_json/app.json`  
  app.json 文件的填写规范详见 PSDK 软件开发包中的`samples/sample_c/platform/linux/manifold3/app_json/README.md`文件或者 <a href="../build-dpk.html#app-json-应用级配置文件说明" target="_blank" rel="noopener noreferrer"> app.json 应用级配置文件说明</a> 章节。

### 开发建议

* **日志文件目录规范**  
建议将妙算应用的日志文件统一写入 `data/logs` 目录。  
否则，应用数据清除、日志导出等功能可能无法正常工作。

* **媒体文件目录规范**  
建议将应用生成或使用的媒体文件存放于 `data/media` 目录。  
否则，应用数据清除及媒体相关功能可能受到影响。
