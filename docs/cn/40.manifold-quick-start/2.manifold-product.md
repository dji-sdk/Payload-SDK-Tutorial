## 妙算3介绍
妙算3定位为大疆行业无人机算力拓展模块，满足行业日益增长的自动化作业和 AI 应用需求，提供 100T 算力资源和原生的 PSDK 接口支持，可跟 DJI 飞行平台实现无缝兼容和通讯，支持获取飞行平台的媒体及传感器数据，支持飞行控制和航线规划，可实现更多自动化业务。开发者只需要专注于 PSDK 软件业务和AI应用的开发，无需担心硬件设计及兼容性问题，全力打造满足行业需求的端侧 AI 应用软件，并可打包成 DPK 格式的软件包分发到全球各地的无人机用户手中，开启您的全新商业模式。

### 查看应用信息
支持在 PILOT 和司空查看妙算3设备上的所有应用信息和单个应用的详细信息。在司空妙算3应用管理界面会显示出所有应用的详细信息，而在 PILOT APP 上，进入到妙算3界面之后，会显示出妙算3设备上的所有应用，如果想要显示单个应用详细信息，需要选择一个应用，点击进入之后便可查看到它的详细信息。

### 三方应用卸载、数据清除、日志导出
支持在 PILOT 和司空上进行三方应用卸载、数据清除和日志导出功能。
- 应用卸载：如果安装/更新完成之后的三方 DPK 应用不想再使用，想要将它从妙算3设备中删除，操作为：进入到妙算3应用管理界面 > 选择想要删除的应用 > 点击应用卸载选项，即可删除对应的应用。
- 数据清除：在三方应用运行过程中会产生日志文件和各类数据，可以清除三方应用运行过程中产生的这些数据，操作为：在妙算3应用管理界面选择数据清除选项。
- 日志导出：进入妙算3应用管理界面之后，可以导出三方应用的日志，以方便开发商进行售后分析问题。
  - 在 PILOT 上导出日志：将 U 盘或读卡器插入妙算的通用 USB 口，之后在 PILOT 上点击“日志导出”选项，即可将妙算上的应用日志导出到 U 盘上。
  - 在司空上导出日志：选择想要导出日志的应用，点击“日志导出”选项，即可导出应用日志到本地。

### 修改开机自启动应用
1. 支持在 PILOT 上进行修改应用开机自启动功能，在 PILOT 上的操作为：点击 妙算3 > 应用管理 > 每个应用的更多 > 设为开机自启动选项，即可修改当前应用为开机自启。
2. 已经设置为自启的应用支持取消开机自启动，操作方式同上。
3. 仅支持一个应用进行开机自启动，后设置的应用拥有开机自启动权。

### 固件升级功能
支持通过 DJI 调参软件和 DJI PILOT APP 升级妙算3固件，推荐使用调参软件升级妙算3固件版本。

### 文件导出功能
| 导出方式                                   |说明|
| :-------------------------------------------------------- |:-------------------------------------- |
| U 盘导出方式                    | 通过 U 盘或者 SSD 固态硬盘连接妙算 USB-C 接口可在遥控器 PILOT APP 上操作一键导出应用数据。|
| 电脑导出方式                   | 妙算挂载在飞机上，通过飞机的调参口连接电脑，再DA2的日志导出界面点击“打开设备目录”，然后在 windows 上通过文件资源管理器导出文件。|
| 调试导出方式（开发者功能）       | 开发者可以直接通过 ssh 和 sftp 导出文件数据|

### 使用 ssh 命令安装、更新、卸载、启停应用等
支持通过 ssh 进入妙算3设备后，使用命令来安装、更新、卸载、启动、停止、重启 DPK 应用，并且查看妙算3设备内应用的信息以及运行状态，具体命令如下：

```bash
# 安装/更新应用
dji_app_ctl install -i xxx.dpk
# 卸载应用
dji_app_ctl uninstall xxx(app_name)
# 启动应用
dji_app_ctl start xxx(app_name)
# 停止应用
dji_app_ctl stop xxx(app_name)
# 重启应用
dji_app_ctl restart xxx(app_name)

# 查看所有应用信息
dji_app_ctl list
# 查看所有应用运行状态
dji_app_ctl status

# 查看所有名为指定应用名称的应用信息
dji_app_ctl list xxx(app_name)
# 查看所有名为指定应用名称的应用运行状态
dji_app_ctl status xxx(app_name)
```
