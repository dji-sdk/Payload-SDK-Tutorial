最新的 DJI PSDK 软件开发包中包含着妙算应用程序的示例代码，可以通过如下步骤编译运行示例程序，来快速了解和体验妙算应用相关功能。

## 1. 修改必要的配置代码

您需要通过开发者用户中心，创建应用后将生成的 Payload SDK-妙算3应用信息，填入前一步获取的 PSDK 软件开发包的指定文件中，否则示例应用程序将无法正常运行。

* samples/sample_c/platform/linux/manifold3/application/dji_sdk_app_info.h
* samples/sample_c++/platform/linux/manifold3/application/dji_sdk_app_info.h

```c
/* Exported constants --------------------------------------------------------*/
// ATTENTION: User must goto https://developer.dji.com/user/apps/#all to create your own dji sdk application, get dji sdk application
// information then fill in the application information here.
#define USER_APP_NAME               "your_app_name"
#define USER_APP_ID                 "your_app_id"
#define USER_APP_KEY                "your_app_key"
#define USER_APP_LICENSE            "your_app_license"
#define USER_DEVELOPER_ACCOUNT      "your_developer_account"
#define USER_BAUD_RATE              "460800"
```
自带的 sample 代码中已经完成妙算 3 硬件链路的代码适配，用户无需做修改。

## 2. 编译和sample体验

PSDK 软件包支持 CMake 编译。请按照以下步骤操作：

1. 修改 `CMakeLists.txt` 文件，将以下内容：

    ```cmake
    add_subdirectory(samples/sample_c/platform/linux/manifold2)
    add_subdirectory(samples/sample_c++/platform/linux/manifold2)
    ```

    修改为：

    ```cmake
    add_subdirectory(samples/sample_c/platform/linux/manifold3)
    add_subdirectory(samples/sample_c++/platform/linux/manifold3)
    ```

2. 随后可以通过 **cmake** 命令进行编译，生成可执行文件。
   编译完成后，`build/` 目录下将会生成可执行文件：

    - `dji_sdk_demo_on_manifold3`
    - `dji_sdk_demo_on_manifold3_cxx`

3. 运行上述可执行文件，即可体验示例代码效果。

```c
cd Payload-SDK/
mkdir cmake-build-debug && cd cmake-build-debug
cmake ../  && make -j
./bin/dji_sdk_demo_on_manifold3
```
