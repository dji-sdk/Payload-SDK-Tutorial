# License 校验指导说明

## 简要说明
为开发的应用 DPK 增加 License 管理机制，防止未经授权的设备使用
1. License 与唯一设备 ID 强绑定，防止复制到其他设备。
2. 支持设定有效期，License 到期后需重新授权。
3. 支持不同的授权校验模式，包括在线校验和离线校验。

## 设备信息获取及操作
1. 获取妙算3的唯一硬件编码  
cat /factory/device_id.txt 或者 cat /sys/firmware/devicetree/base/serial-number
2. 获取时间
可通过 PSDK 的数据订阅获取飞机的 GPS UTC 时间
3. 用户界面输入
可使用 PSDK 提供的 Pilot 控件，支持用户在遥控器 Pilot 或者司空2界面上输入文本字符的激活码

## 存储与安全建议
1. License 文件存储在应用私有目录，防止被用户复制替换，参考路径 "open_app/xxx 应用/xxx 自定义目录" 或者  "/home/dji/xxx 自定义目录"
2. 文件加密或混淆处理，防止直接读取或修改。
3. 算法核心逻辑进行代码混淆、防逆向（如 LLVM 混淆）。


## 方案选型建议
1. 在线模式（非对称算法）：  
优点：安全性较高，私钥不在设备端。  
缺点：依赖网络，用户离线环境下不可用。

2. 离线模式（对称/私有算法）：  
优点：适合离线使用环境，可本地完成激活与验证。  
缺点：安全性取决于算法保护强度，需要防止逆向。
