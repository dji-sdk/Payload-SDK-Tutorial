## 什么是妙算应用（DPK）？
妙算应用是基于 DJI PSDK 接口为妙算 Manifold 3 开发的应用软件，简称为妙算应用（Manifold Application），后称 DPK 应用。

开发者通过交叉编译的方式，直接使用 DJI 提供的应用打包工具 build_dpk.sh 可打包生成一个名为 xxx.dpk 后缀的安装包（即妙算应用安装包），最终用户可以通过 DJI Pilot APP 或者妙算3设备内的应用管理工具 dji_app_ctl 来安装、更新和卸载妙算应用，极大降低用户的使用门槛（相比妙算2无需用户学习 linux 相关指令，降低用户现场的部署难度）。

## 怎么开发妙算应用？
开发妙算应用时，需要遵循如下开发约束，并且推荐按照下面的开发建议：
### 约束
* 三方依赖库需要使用静态依赖库，不支持使用动态依赖库，如果使用动态库可能会出现妙算应用安装或者启动失败的问题。
* 源码中读写配置文件时，配置文件的路径需要写成相对路径，不能写成绝对路径，否则可能会出现安装或者启动妙算应用失败的问题。
* 需要参考 DJI PSDK 软件开发包中的`samples/sample_c/platform/linux/manifold3/app_json/app.json`，建立自己的应用的 app.json 文件，app.json 要求严格按照规范来填写，开发者网站上生成的 APP ID 写到文件中的 user_app_id 字段，代码中的 firmwareVersion 应用版本信息写到 firmware_version 字段，编译生成的二进制文件写到 bin 字段，配置文件写到 userconfig 字段......，否则会生成 xxx.dpk 应用安装包失败或者安装应用失败，app.json 文件的填写规范详见 PSDK 软件开发包中的`samples/sample_c/platform/linux/manifold3/app_json/README.md`文件或者 <a href="./build-dpk.html#app-json-应用级配置文件说明" target="_blank" rel="noopener noreferrer"> app.json 应用级配置文件说明</a> 章节。

### 建议
* 妙算应用日志文件写到 data/logs 目录中，否则应用数据清除和应用日志导出功能可能会受到影响。
* 妙算应用媒体文件写到 data/media 目录中，否则应用数据清除和有关媒体的其他功能可能会受到影响。
