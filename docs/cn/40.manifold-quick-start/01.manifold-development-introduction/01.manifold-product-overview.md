## 什么是妙算3
妙算 3 是一款面向 DJI 行业无人机的算力拓展模块，提供高性能端侧算力资源与原生 PSDK 接口支持，可与 DJI 飞行平台实现无缝兼容与稳定通信，支持获取飞行平台的媒体与传感器数据，并结合飞行控制与航线规划能力，帮助开发者构建自动化与智能化应用。	妙算 3 对底层硬件和系统能力进行统一封装，使开发者能够专注于 PSDK 软件与 AI 应用开发，并将应用打包为 DPK 格式进行部署和分发，从而高效实现行业级应用落地。
## 核心理念
妙算开发致力于将复杂的计算与数据处理能力进行模块化、组件化封装，让开发者无需关注底层实现细节，即可快速调用强大的算力能力。 其核心理念主要体现在以下几个方面：
- 算力即服务：计算能力以服务形式提供，即开即用。
- 低门槛开发：配置化、模板化，减少代码编写。
- 模块化组合：能力解耦，灵活拼装。

## 妙算3架构概览

妙算3是一款高性能机载算力平台，具备与飞行平台深度交互的能力，可实现飞行控制与遥测数据获取，同时能够控制和管理有效载荷，包括 DJI 官方负载及 PSDK 生态负载。妙算3可通过 DJI SDR/4G 增强图传链路与 DJI 遥控器上的 MSDK 应用、Pilot 实现互联通信；也支持通过 Pilot 上云或机场上云，与第三方云平台及司空2实现互联通信。此外，妙算3还可借助机载4G 或扩展的4G/5G CPE 网卡建立自定义通信链路。  
妙算3进一步整合了 PSDK 应用、PSDK 负载设备、基于 MSDK 的移动端 App，以及基于上云 API 的云平台，实现各节点间的互联互通，构建起完整的端到端业务交互体系。
