
## 升级固件
固件升级方式：推荐使用 DJI Assistant 2 (行业系列) 调参软件进行固定升级和更新。最新固件版本详见 [版本发布记录](https://developer.dji.com/doc/payload-sdk-tutorial/cn/)


## 访问妙算 3

妙算 3 已内置 PSDK 开发所需的软件环境，用户登录后即可直接进行应用开发。

**登录控制台或者 GUI 方式：**
- SSH（推荐，支持 Windows 终端和 VSCode）
- VNC 远程桌面（GUI）

**传输文件，支持以下两种方式：**
- SCP（Windows 系统推荐使用 WinSCP）
- Samba 共享（Windows 系统通过 DJI Assistant2 打开共享目录）

### SSH 登录

如果通过 USB 线将 PC 与飞行器调参接口直连，可根据妙算 3 的挂载位置，使用以下 IP 地址进行访问：

| 飞行平台 | 接口编号 | SSH 地址 |
| :-- | :-- | :-- |
| Matrice 4E/4T/4TD/4D | E-Port | `192.168.42.120` |
| Matrice 400 | E1 | `192.168.42.120` |
| Matrice 400 | E2 | `192.168.42.130` |
| Matrice 400 | E3 | `192.168.42.140` |

如果通过妙算 3 的 USB 拓展坞外接 USB 转以太网网卡或 Wi‑Fi 网卡，则可登录所接入局域网的路由器后台，查看设备分配到的 IP 地址。

**默认登录信息：**
- 默认用户名：`dji`
- 密码：`dji`

#### Windows 终端
Windows 11 的用户可以通过 Windows 终端访问妙算 3。在 Windows 终端中新建配置项，名称可自定义，如 `Manifold3`。  
命令行修改为：

```bash
ssh dji@192.168.42.140
```

其中 `192.168.42.140` 为妙算 3 的 IP 地址。保存后再次打开，输入密码后即可访问。

## 交叉编译

为提升开发和调试效率，用户可以在 Ubuntu PC 上安装工具链，或使用 [NVIDIA 交叉编译工具链](https://developer.nvidia.com/embedded/jetson-linux-archive) 进行开发和编译。  
编译完成后，可通过 SSH 将可执行文件推送至 Manifold 3 进行调试。  
调试通过后，可将应用打包为 DPK 格式，便于在更多设备上安装和使用。

::: warning 注意：

妙算3设备内置 GCC 版本为9.4.0，若安装的交叉编译工具链高于此版本，可能会引入潜在的不兼容问题。
:::


### 交叉编译构建 PSDK 可执行文件
下载所需的 <a href="https://github.com/dji-sdk/Payload-SDK/releases" target="_blank">PSDK 版本</a> ，在 PSDK 文件夹内打开终端，输入：

```bash
mkdir build && cd build
cmake .. -DCMAKE_C_COMPILER=aarch64-linux-gnu-gcc -DCMAKE_CXX_COMPILER=aarch64-linux-gnu-g++
make
```
编译完成后，bin 目录下将生成可在 Manifold 3 上运行的可执行文件   `dji_sdk_demo_on_manifold3 `、 `dji_sdk_demo_on_manifold3_cxx `。
