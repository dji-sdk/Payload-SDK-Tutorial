**连接方式1：** 妙算3使用 Matrice 400配件包连接 Matrice 400飞机，用户可以通过 USB 数据线直连 PC 和飞机调参口，通过 USB 虚拟网卡访问不同接口对应的 IP 地址访问妙算3。
- 当妙算3连接 Matrice 400 E1，访问妙算3的 IP 地址：192.168.42.120
- 当妙算3连接 Matrice 400 E2，访问妙算3的 IP 地址：192.168.42.130
- 当妙算3连接 Matrice 400 E3，访问妙算3的 IP 地址：192.168.42.140

## 妙算3连接外部网络
该连接方式为以上 Matrice 400、Matrice 4D/4TD、Matrice4E/4T 开发环境拓扑图中的连接方式2，妙算3 使用 USB 拓展坞外接 PD 供电和网卡，用户可以通过网线、USB WiFi 接入局域网后根据路由器分配的 IP 地址登录访问妙算3。
- 首次访问：可以通过路由器后台或者连接方式1查看妙算3的 IP。
- 设置固定 IP：通过路由器设置固定 IP 或者使用妙算3的 <a href="../manifold-platform-capabilities/system-tools.html#网络配置工具" target="_blank" rel="noopener noreferrer">netctl.sh 网络配置</a>。

## 妙算3与 PC 直连
妙算3接入 USB 拓展坞或者 USB 以太网模块，通过 RJ45网线直连 Windows PC 网口（需要 PC 具备2个以太网接口，或者 PC 通过 WLAN 访问互联网）。
在 windows 中打开，控制面板 > 网络和 Internet > 网络连接。
<div  align=left>
<img src="https://terra-1-g.djicdn.com/71a7d383e71a4fb8887a310eb746b47f/psdk/%E5%A6%99%E7%AE%97%E5%BC%80%E5%8F%91/%E7%A1%AC%E4%BB%B6%E7%8E%AF%E5%A2%83%E6%90%AD%E5%BB%BA/%E5%A6%99%E7%AE%973%E4%B8%8EPC%E7%9B%B4%E8%BF%9E.png"  style="width: 50%;" />
</div>

如图所示，至少可以看到两个网络适配器，其中一个是 PC 电脑正常访问互联网的网络适配器，另外一个是连接妙算3的网络适配器。

<div  align=left>
<img src="https://terra-1-g.djicdn.com/71a7d383e71a4fb8887a310eb746b47f/psdk/%E5%A6%99%E7%AE%97%E5%BC%80%E5%8F%91/%E7%A1%AC%E4%BB%B6%E7%8E%AF%E5%A2%83%E6%90%AD%E5%BB%BA/%E4%BB%A5%E5%A4%AA%E7%BD%91%E5%85%B1%E4%BA%AB.png"  style="width: 30%;" />
</div>


通过以太网共享，可实现妙算 3 与 PC 之间的通信，并为妙算 3 提供互联网访问能力。请右键点击 PC 上已联网的网络适配器 - 选择共享 - 允许共享 - 选择连接妙算 3 的网络适配器。

<div  align=left>
<img src="https://terra-1-g.djicdn.com/71a7d383e71a4fb8887a310eb746b47f/psdk/%E5%A6%99%E7%AE%97%E5%BC%80%E5%8F%91/%E7%A1%AC%E4%BB%B6%E7%8E%AF%E5%A2%83%E6%90%AD%E5%BB%BA/%E6%9F%A5%E7%9C%8BIP.png"  style="width: 40%;" />

设置完成后，即可在连接妙算 3 的网络适配器上看到已分配的 IP 地址 192.168.137.1。
