## 硬件环境准备

妙算3提供两种供电方式：
连接飞机(M400 飞机 E-Port V2接口/Matrice 4D 系列飞机E-Port 接口)供电，或使用 USB 拓展坞 PD 供电口接入电源适配器（65W/100W）。<br>
>- 妙算3的 USB-C 接口支持连接拓展坞进行开发环境的搭建（不支持显示输出），参考型号：绿联 CM512Type-C 扩展坞、安克（A8380）553PowerExpand 8-in-1 USB-C 集线器。
>- Matrice 4D/4TD系列飞机在连接方式1下，必须先完成与妙算3的接线后才能启动飞机，该连接方式不支持热插拔。

使用 E-Port V2 连接飞机时，用户可以通过 USB 线直连 PC 和飞机调参口，通过 USB 虚拟网卡访问不同接口对应 IP 地址访问妙算3。
使用 USB 拓展坞连接时，可以通过网线接入局域网后根据路由器分配的 IP 地址 ssh 登录访问妙算3。
<br>两种方式下均支持用户使用 ssh 或 VNC 访问妙算3，进行开发调试。
