## 妙算开发核心理念
妙算开发致力于将复杂的计算与数据处理能力进行模块化、组件化封装，让开发者无需关注底层实现细节，即可快速调用强大的算力能力。
其核心理念主要体现在以下几个方面：
- 算力即服务：计算能力以服务形式提供，即开即用。
- 低门槛开发：配置化、模板化，减少代码编写。
- 模块化组合：能力解耦，灵活拼装。

## 硬件环境准备
妙算 3 提供以下两种供电与连接方式：
1. 连接飞机供电
   - Matrice 400：E-Port V2接口
   - Matrice 4D：E-Port 接口
2. 通过 USB 拓展坞 PD 供电口接入电源适配器（65W/100W）。<br>

::: tip 说明：
- 妙算3的 USB-C 接口支持连接拓展坞进行开发环境的搭建（不支持显示输出），参考型号：绿联 CM512Type-C 扩展坞、安克（A8380）553PowerExpand 8-in-1 USB-C 集线器。
- Matrice 4D/4TD 系列飞机在连接方式1下，必须先完成与妙算3的接线后才能启动飞机，该连接方式不支持热插拔。
- 使用 E-Port V2 连接飞机时，用户可以通过 USB 线直连 PC 和飞机调参口，通过 USB 虚拟网卡访问不同接口对应 IP 地址访问妙算3。
- 使用 USB 拓展坞连接时，可以通过网线接入局域网后根据路由器分配的 IP 地址 ssh 登录访问妙算3。
- 两种方式下均支持用户使用 ssh 或 VNC 访问妙算3，进行开发调试。
::: 

> 提示：首次体验用户强烈建议使用「USB 拓展坞供电 + 局域网 + SSH 登录」方式，路径最简单、问题最少。

## 开发环境准备
### 登录妙算 3
妙算 3 已内置 PSDK 开发所需的软件环境，用户登录后即可直接进行应用开发。
支持以下两种登录方式：
- SSH（推荐）
- VNC 远程桌面

以下以 SSH 登录 为例进行说明。  
**SSH 登录信息（Matrice 400）**

| E-Port 编号|ssh 地址|
| :---- | :---- |
|E1|192.168.42.120|
|E2|192.168.42.130|
|E3|192.168.42.140|


默认登录信息：
- 默认用户名：dji
- 密码：dji

### 安装交叉编译工具链

为提升开发和调试效率，用户可以在 Ubuntu PC 上安装工具链，或使用 [NVIDIA 交叉编译工具链](https://developer.nvidia.com/embedded/jetson-linux-archive) 进行开发和编译。  

::: warning 注意：

妙算3设备内置 GCC 版本为9.4.0，若安装的交叉编译工具链高于此版本，可能会引入潜在的不兼容问题。
:::

## 运行 Demo
最新的 [DJI PSDK 软件开发包](https://github.com/dji-sdk/Payload-SDK/releases) 中包含着妙算应用程序的示例代码，可以通过如下步骤编译运行示例程序，来快速了解和体验妙算应用相关功能。

### 修改必要的配置代码

将获取的[**应用信息**](#获取应用信息)，填入 `dji_sdk_app_info.h` 文件中，否则示例应用程序将无法正常运行。

* samples/sample_c/platform/linux/manifold3/application/dji_sdk_app_info.h
* samples/sample_c++/platform/linux/manifold3/application/dji_sdk_app_info.h

```c
/* Exported constants --------------------------------------------------------*/
// ATTENTION: User must goto https://developer.dji.com/user/apps/#all to create your own dji sdk application, get dji sdk application
// information then fill in the application information here.
#define USER_APP_NAME               "your_app_name"
#define USER_APP_ID                 "your_app_id"
#define USER_APP_KEY                "your_app_key"
#define USER_APP_LICENSE            "your_app_license"
#define USER_DEVELOPER_ACCOUNT      "your_developer_account"
#define USER_BAUD_RATE              "460800"
```
自带的 sample 代码中已经完成妙算 3 硬件链路的代码适配，用户无需做修改。

### 编译和 Demo 体验

PSDK 软件包支持 CMake 编译。请按照以下步骤操作：

1. 修改 `CMakeLists.txt` 文件，将以下内容：

    ```cmake
    add_subdirectory(samples/sample_c/platform/linux/manifold2)
    add_subdirectory(samples/sample_c++/platform/linux/manifold2)
    ```

    修改为：

    ```cmake
    add_subdirectory(samples/sample_c/platform/linux/manifold3)
    add_subdirectory(samples/sample_c++/platform/linux/manifold3)
    ```
2. 通过 **cmake** 命令进行编译，在 PSDK 文件夹内打开终端，输入：  
   
   ```bash
   mkdir build && cd build
   cmake .. -DCMAKE_C_COMPILER=aarch64-linux-gnu-gcc -DCMAKE_CXX_COMPILER=aarch64-linux-gnu-g++
   make
   ```
   编译完成后，`build/` 目录下将会生成可执行文件：

    - `dji_sdk_demo_on_manifold3`
    - `dji_sdk_demo_on_manifold3_cxx`

3. 运行上述可执行文件，即可体验示例代码效果。

   ```c
   cd Payload-SDK/
   mkdir cmake-build-debug && cd cmake-build-debug
   cmake ../  && make -j
   ./bin/dji_sdk_demo_on_manifold3
   ```
