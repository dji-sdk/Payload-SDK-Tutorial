## 概述

基于第三方PSDK负载搭载DJI 空吊2.0系统开发，可以实行对空吊系统的简单控制，并且获取空吊系统的实时状态反馈。

### 在 Flycart100上使用

* 使用对象：1 个 E-Port Lite 口（PSDK）、一台Flycart机型
* 使用范围：通过 E-Port Lite 口（PSDK）可以获取空吊系统的状态
* 使用约束：不支持负载空吊控制和复杂的状态获取
  
## PSDK控制空吊系统功能介绍
使用PSDK可以对空吊系统进行简单的控制。**基于PSDK的功能开发，实现对DJI空吊的控制。**

## 使用PSDK控制空吊功能功能 

#### 1. 空吊控制初始化模块初始化
使用“空吊控制”功能前，需要先初始化功能模块，确保功能可正常运行。  

```c
returnCode = DjiAircraftInfo_Init();
if (returnCode != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS) {
    DJI_ERROR_PRINT_MSG(TAG_NAME_CORE, returnCode);
    DJI_LOG_ERROR(TAG_NAME_CORE, "Aircraft info init error");
    return returnCode;
}
```

#### 确认机型是否正确
**建议**通过PSDK lib里面提供的机型获取函数获取机型。

```
returnCode = DjiAircraftInfo_GetBaseInfo(&aircraftBaseInfo);
if (returnCode != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS) {
    DJI_LOG_ERROR(TAG_NAME_CORE, "Get aircraft base info error");
    return returnCode;
} else {
    DJI_LOG_INFO(TAG_NAME_CORE, "Identify AircraftType = %s, MountPosition = %s, SdkAdapterType = %s",
                    DjiAircraftInfo_GetAircraftName(aircraftBaseInfo.aircraftType),
                    DjiAircraftInfo_GetMountPositionName(aircraftBaseInfo.mountPosition),
                    DjiAircraftInfo_GetAdapterName(aircraftBaseInfo.djiAdapterType));
}
```

#### 2. 开启实时获取空吊系统状态和空吊的实时控制
开启task运行，目前建议不高于10HZtask运行

1. 请求飞机控制空吊系统（地面无法控制，需要APP模拟器起飞）     

```c
djiStat = psdk_HoistController_NotifyFlight_to_startPushInfo();
if (djiStat != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS) {
    USER_LOG_ERROR("get hoist can start contrller error: 0x%08llX.", djiStat);
    continue;
}
```

2. 开始控制空吊系统

空吊卷扬机控制接口：
```c
T_DjiReturnCode psdk_hoistControllerCmdAction(E_DjiHoistControllerCmdStatus cmd)
```

空吊主动挂钩控制接口：
```c
T_DjiReturnCode psdk_hookControllerCmdAction(E_DjiHookControllerCmdStatus cmd)
```

3. 获取空吊返回状态

```c
static T_DjiReturnCode subscriptionHoistAndHookStatusData(void)
{
    T_DjiReturnCode djiStat;
    static uint32_t cnt = 0;

    cnt++;
    djiStat = DjiHoistController_hoistStatus(&hoist_status); //空吊卷扬机状态
    if (djiStat != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS){
        return djiStat;
    }

    if (cnt % 5 == 0){
        USER_LOG_INFO("hoist status:cab_s:%d ,can_s:%d, g_w:%d, mg_w:%d, c_s_s:%d, cc_l:%d",
            hoist_status.cable_state, hoist_status.can_status, hoist_status.goods_weight, hoist_status.max_goods_weight,
            hoist_status.cargo_swing_speed, hoist_status.cur_cable_length);
        USER_LOG_INFO("hoist status: c_f:%d, c_an:%d, is_r_t:%d, is_rmr:%d, cdcs:%d",
            hoist_status.cable_force, hoist_status.cable_angle, hoist_status.is_reach_top, hoist_status.is_reach_max_round,
            hoist_status.cable_direction_check_state);
    }

    djiStat = DjiHoistController_hookStatus(&hook_status); //空吊主动挂钩状态
    if (djiStat != DJI_ERROR_SYSTEM_MODULE_CODE_SUCCESS){
        return djiStat;
    }
    if (cnt % 5 == 0){
        USER_LOG_INFO("hook status: steer_state:%d, weight:%d, warning:%d",
            hook_status.steer_state, hook_status.cargo_weight, hook_status.cargo_warnig);
        USER_LOG_INFO("hook status: error:%d, ctr_state:%d", hook_status.cargo_error, hook_status.ctrl_state);
    }

    return djiStat;
}
```
> **说明：** 目前仅 Flycart 100 支持。
