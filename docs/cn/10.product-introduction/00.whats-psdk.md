PSDK（Payload SDK）是一套用于扩展无人机负载能力的软件开发接口，支持开发者将自研或第三方负载接入无人机平台，实现控制、数据交互及系统协同。为方便您使用 PSDK 提供的新功能开发出安全可靠的负载设备，请持续关注 DJI PSDK 的版本发布信息，及时使用[最新版本的 PSDK 开发包](https://github.com/dji-sdk/Payload-SDK/releases)开发负载设备。
:::tip 本文所指 
* MSDK 是一款软件开发套件，旨在让开发者能够访问 DJI 无人机和手持相机产品的丰富功能。开发者可以使用 MSDK 开发的移动端 App 控制负载设备执行指定的动作和任务。点击 [更多](https://developer.dji.com/doc/mobile-sdk-tutorial/cn/) 了解 MSDK 的更多功能。
* “移动端 App” 为使用 MSDK 开发的移动端 App 或 DJI Pilot。
* “负载设备” 为使用 PSDK 开发的负载设备。
:::



## PSDK 介绍
DJI 为支持开发者开发出可挂载在 DJI 无人机上的负载设备，提供了以 PSDK 为核心的软件接口体系，并配套 [X-Port 标准云台](https://store.dji.com/cn/product/dji-x-port)、[SkyPort V3 转接环](https://store.dji.com/cn/product/dji-skyport-v3-adapter-set?vid=192891) 和 [E-Port V2同轴线](https://store.dji.com/cn/product/dji-e-port-v2-coaxial-cable-kit?vid=192711) 等硬件组件，帮助开发者更高效地使用 DJI 无人机上如电源、通讯链路及状态信息等**资源**。开发者能够根据行业的应用需求，基于 PSDK 提供的功能接口，结合具体的结构设计、硬件设计、软件逻辑实现和算法优化，开发出如**自动巡检系统**、**红外相机**、**测绘相机**、**多光谱相机**、**喊话器**、**探照灯**等满足不同细分领域的负载设备。

## PSDK 在系统中的角色与使用方式
在实际应用中，无人机系统通常由飞行平台、负载设备、机载计算平台及地面应用协同组成。
PSDK 作为负载能力接入与扩展的核心接口，可应用于不同的开发形态中，用于实现负载与无人机平台之间的通信、控制及数据交互。
根据应用运行位置的不同，PSDK 通常可用于以下两种开发方式： 
- 负载开发：在负载设备上直接使用 PSDK 实现负载本体功能；
- 妙算开发：在妙算机载计算平台上使用 PSDK，与负载设备和飞行平台进行协同。

此外，PSDK 还可与 Mobile SDK（MSDK）配合使用，由移动端 App 实现负载控制与任务管理，形成完整的行业解决方案。

## 主要优势
* **支持多种开发形态**<br/>
PSDK 可灵活应用于妙算开发与负载开发场景，满足不同算力需求和系统架构选择。
* **统一的负载接入能力**<br/>
通过统一接口规范，降低不同负载类型的接入成本。

* **功能丰富且完善**<br/>
  通过使用 PSDK 提供的如信息获取、数据传输和电源管理等基础功能，以及相机、云台、负载协同和精准定位等高级功能，开发者能够根据行业的应用需求，设计出**功能完善**的负载设备。并且 PSDK 提供了丰富的接口，方便开发者使用第三方应用程序和算法框架，使用图像识别、自主巡航及 SLAM 等技术开发出专业的应用软件，此外，还方便开发者接入第三方传感器、相机或检测设备，采集所需的数据信息，满足用户个性化的应用功能和控制需求。

* **拓展应用自定义**<br/>
  开发者除使用 DJI Pilot 控制基于 PSDK 开发的负载设备和控制无人机和负载设备**自动执行任务**的控制程序外，通过使用 Mobile SDK 也能开发出控制负载设备执行指定动作的**移动端 App**，使用 Windows SDK 还能够开发出数据处理软件，此外，开发者还能将所开发的软件集成为一套**功能完整的解决方案**，满足第三方用户丰富的使用需求。

* **兼容广泛的软硬件平台**<br/>
  使用 PSDK 开发的应用程序能够运行在主流的嵌入式硬件平台上，如 STM32等，也可运行在主流的嵌入式操作系统和软件架构中，如 Linux、RTOS，开发者按照<a href="../payload-quick-start/getting-started-with-psdk.html#跨平台移植" target="_blank" rel="noopener noreferrer">跨平台移植</a>中的步骤执行所需的工作后，即可在不同版本的软硬件平台上运行基于 PSDK 开发的应用程序。

* **支持服务有保障**<br/>
  PSDK 不仅提供了用于开发 DJI 无人机负载设备的 API 接口和硬件平台，还提供了开发负载设备的**设计标准**，此外，还提供了包括但**不限于**技术支持、负载检测及市场推广等服务，服务开发者使用 PSDK 开发出功能完善的无人机负载设备，探索行业应用的无限潜能。
