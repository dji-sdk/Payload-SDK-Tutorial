## 开发与量产
### 开发前准备
使用 PSDK 开发负载设备前，建议先学习开发负载设备所需的基础知识，了解 PSDK 的功能以及负载设备所需满足的标准。
* 学习无人机基础知识和控制原理：如俯仰、偏航、滚转和升降等基础知识
* 查看 DJI 的
  <a href="../payload-quick-start/configure-the-hardware-environment/payload-develop-criterion.html" target="_blank">负载标准</a>

### 开始开发负载设备
使用 PSDK 开发负载设备时，**请正确地连接**所选购的硬件平台、第三方开发板和 DJI 的无人机，**正确地配置**负载设备开发环境，正确导入 PSDK 并完成初始化配置。
* 在使用 PSDK 开发负载设备前，请先阅读
  <a href="../development-preparation/safety-information.html" target="_blank">安全须知</a>
  中的内容，避免因操作不当损毁负载设备或无人机。
* 连接无人机、硬件平台、第三方开发板和计算机。
  * <a href="../payload-quick-start/configure-the-hardware-environment/device-connect.html" target="_blank">负载开发硬件连接</a>
  * <a href="../manifold-quick-start/manifold-connect.html" target="_blank">妙算开发硬件连接</a>
* 配置开发环境，准备相关工具链和库。
  * <a href="../payload-quick-start/config-develop-environment.html" target="_blank">负载开发配置开发环境</a>
  * <a href="../manifold-quick-start/manifold-development.html" target="_blank">妙算开发配置开发环境</a>
* 导入 PSDK，完成初始化。
  * <a href="../payload-quick-start/getting-started-with-psdk.html" target="_blank">负载开发导入 PSDK</a>
  * <a href="../manifold-quick-start/getting-started-with-psdk.html" target="_blank">妙算开发导入 PSDK</a>

### 开发功能完善的负载设备
* 请根据
  <a href="../function-overview/basic-function/log-management.html" target="_blank">功能介绍</a>
  和
  <a href="https://developer.dji.com/doc/payload-sdk-api-reference/cn/" target="_blank">PSDK API 文档</a>
  ，开发负载设备。
* 借助 [DJI Assistant 2](https://www.dji.com/cn/downloads) 等工具调试负载设备，完善负载设备的功能，提高负载设备的性能。

### 申请负载认证，开通量产功能
进入 DJI 开发者网站，点击 个人中心 -> 应用详情 -> 应用认证。
:::warning 注意：
* 1 个 PSDK APP 应用授权可绑定负载的数量上限为 20 个，申请开通量产功能后将会解除该限制。
* 自**2025年1月1号起**，新注册的 PSDK 应用默认为开发模式，需要搭配 **Skyport V2, X-Port, SDK 认证芯片**进行绑定的生态负载在开发模式下**均有该数量限制**, 不涉及绑定流程的生态负载不受限制。
:::

## 成为 DJI 的合作伙伴
DJI 秉承**开放共赢**的合作理念，为开发者提供技术支持、负载检测、渠道推荐以及 DJI 生态推广等服务，诚挚携手广大开发者以及合作伙伴共建可持续发展的应用生态。

* 若合作伙伴开发的负载设备能通过 DJI 的负载检测，该负载设备将被 DJI 推荐给第三方合作伙伴。
* 若合作伙伴的负载设备能够通过第三方专业机构的检测，在与 DJI 签署合作协议后，DJI 将与合作伙伴开展更为深入的合作，如将合作伙伴的负载设备推荐给 <a href="https://www.dji.com/cn/products/enterprise#partner-payloads" target="_blank">全球用户</a> 等，更多合作事宜请<a href="mailto:dev@dji.com"> 联系我们</a>。
