## 什么是 PSDK 开发？
PSDK（Payload SDK）是大疆为第三方开发者提供的开发套件，用于扩展 DJI 飞行平台的功能。通过 PSDK，开发者可以将自研硬件或智能算法深度集成至无人机系统，实现数据采集、智能分析与自动控制等能力。借助 PSDK，开发者可以：
- 开发可挂载于无人机上的硬件负载设备
- 开发运行于机载计算平台的智能软件应用
- 深度获取飞行状态、传感器数据，并控制云台、负载等系统资源

## PSDK 软件框架概览
```
应用层（开发者业务代码）
   ↓
PSDK 核心层（通信 / 设备 / 数据管理）
   ↓
平台层（负载平台 / 妙算平台）
```
- 应用层：实现具体业务逻辑，如算法、控制策略、数据处理。
- PSDK 核心层：负责设备管理、数据通信与接口封装。
- 平台层：对接 DJI 飞行平台或妙算计算硬件。

## 两种开发方式
### 负载开发
负载开发指基于 PSDK 开发可挂载于 DJI 飞行平台的外部物理设备，如相机、传感器、机械装置等。   

**主要形式**
- 负载定制：为飞行平台全新设计和定制开发的硬件类负载产品。
- 负载适配：将现有智能负载迁移至新机型，或将老飞机上的既有负载适配至新飞机平台。

**典型特点**  
- 强调硬件设计与系统集成
- 可直接参与环境感知或执行动作
- 适合对硬件形态或功能有定制需求的项目

### 妙算开发  
妙算开发是指在机载计算平台上开发运行智能软件，实现数据分析与决策能力。

**应用场景**  
- AI 目标识别与分类
- 实时数据分析
- 智能跟踪与决策控制  

**典型特点**  
- 以软件和算法为核心
- 依托机载算力完成实时处理
- 开发周期短、迭代灵活 

## 负载开发与妙算开发的关系
### 1. 互补关系
- **负载开发**：提供"眼睛和手"，负责感知与执行。  
- **妙算开发**：提供"大脑"，负责分析与决策。
- 两者结合可实现完整智能系统。

### 2. 协同工作模式
```
负载设备（如相机）
   ↓ 采集数据
妙算平台（分析处理）
   ↓ 发出指令
负载设备（执行动作）
```
例如：相机采集图像 → 妙算平台进行目标识别 → 输出控制指令 → 云台或机械装置执行动作。
### 3. 实际应用示例
- **农业巡检**：多光谱相机负载 + 病害识别算法
- **安防巡逻**：热成像负载 + 人员检测算法
- **物流配送**：智能货舱负载 + 路径规划算法

## 如何选择开发方式？
### 选择建议
- **以硬件创新为主** → 负载开发
- **以算法和软件为主** → 妙算开发
- **构建完整智能系统** → 负载开发 + 妙算开发

### 快速对比

|   对比项 | 定制负载 | 负载适配 |妙算开发 | 组合方案 |
|:----------:|:----------:|:----------:|:----------:|:----------:|
| **本质** | 硬件产品开发 |软件及驱动开发| 软件功能开发 | 硬件 + 智能软件（即定制负载+妙算） |
| **周期** | 6-12个月 | 1-3个月 |1-2个月 | 6-12个月 |
| **开发成本** | 中高 |中 | 低 | 高 |
| **效果** | 专用功能 | 设备兼容 |智能处理 | 完整行业方案 |

::: warning 注意事项:
以上周期与成本为常见项目经验值，实际情况受功能复杂度、认证要求、团队经验等因素影响。
:::
